# 문제 5.2 임의의 정수값 증가시키기
### Q. 십진수 D를 나타낸 배열 A가 주어졌을 때, D+1의 결과를 다시 배열 A에 갱신하는 코드를 작성하라.
  * 입력 D: `<1,2,9>` -> D+1: `<1,3,0>`

### A1. 단순한 방법
* 배열의 숫자를 하나씩 읽어 하나의 숫자로 만든 후 1 증가시키고 다시 배열로 만들기
* 오버플로우 문제 발생 가능성

### A2. 배열의 숫자에 직접 연산
* 맨 뒤 부터 각 자리에 +1 후 올림 수 처리

#### 복잡도
* 배열의 길이가 `n`이라고 할 때, 시간 복잡도 `O(n)`
