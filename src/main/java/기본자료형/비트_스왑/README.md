# 문제 4.2 비트 스왑

### Q. 64비트로 정수가 주어졌을 때 i번째 비트와 j번째 비트를 스왑하는 코드를 작성하라.
* ex) 8비트 정수 73의 1번 인덱스 비트와 6번 인덱스 비트를 스왑하는 경우
  * 0**1**0010**0**1 (= 73)   
    => 0**0**0010**1**1 (= 11)

### A1. 무식한 방법
* i번째 비트와 j번째 비트를 저장해 둔 후, 비트 마스크와 비트 연산을 통해 서로 상대의 위치에 쓰기

### A2. 개선된 방법
* 두 비트를 비교하여 다를 경우 각 비트를 뒤집기
