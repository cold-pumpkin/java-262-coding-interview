# 문제 4.7 x^y 계산하기
### Q. 실수 x와 정수 y가 주어졌을 때 `x^y`를 계산하는 프로그램을 작성하라.
* 오버플로우, 언더플로우는 무시

### A1. 단순한 방법
* x에 x를 y-1번 곱

#### 복잡도
* 정수의 길이를 n이 라고 할 때 `O(2^n)`

### A2. 곱셈 연산 줄이기
* y가 음수인 경우 `(1/x) ^ (-y)` 로 변경한 후 연산
* y가 0 이상, 짝수인 경우 : `x^(y/2) * x^(y/2)`
* y가 0 이상, 홀수인 경우 : `x * x^(y/2) * x^(y/2)`

#### 복잡도
* 시간 복잡도 `O(n)`
